Faces2Keywords

Faces2Keywords imports Contacts from Google Picasa (found using "face detection") into Adobe Photoshop Lightroom as Keywords.

Visit the web site http://sites.google.com/site/mybyways/faces2keywords for more information. If you like Faces2Keywords, please consider a small donation to keep development going! Thanks!

Installation Instructions

To install:

  1. Unzip the folder Faces2Keywords.lrplugin using a tool like 7-zip
  2. Place the folder in folder of your choice, e.g.
    • C:\Program Files\Adobe\Adobe Photoshop Lightroom 3\
    • C:\Documents and Settings\<username>\Application Data\Adobe\Lightroom\
  3. From Lightroom, select the menu File > Plug-in Manager...
  4. In the Plug-in Manager dialog, click the Add button
  5. Browse to the folder Faces2Keywords.lrplugin and click Ok
  6. The Faces2Keywords Settings page will be displayed

To uninstall:

  1. Select Faces2Keywords in the Plug-in Manager and click Remove
  2. Delete the folder Faces2Keywords.lrplugin

Setup Instructions

Faces2Keywords will try to find Picasa's Contacts.xml file. Faces2Keywords cannot find this file automatically on non-English Windows or MacOS. Instead:

  • Click the ... button next to the Picasa Contacts.xml input field
  • Navigate to the Contacts.xml file and click Ok, e.g.
    • On Windows XP, C:\Documents and Settings\<username>Local Settings\Application Data\Google\Picasa2\contacts\contacts.xml
    • On Windows Vista/7, C:\Users\<username>\AppData\Local\Google\Picasa2\contacts\contacts.xml
    • On Mac OS, /Users/<username>/Library/Application Support/Google/Picasa3/contacts/contacts.xml

Other settings are:

  • Select Use Contact Nickname as Keyword Name to use Picasa's nickname field as the Keyword Name, or
  • Select Use Contact Name as Keyword Name to use Picasa's name field as the Keyword Name
  • And optionally check Use Contact Name/Nickname as Synonym to use the other field (name or nickname) as the Keyword Synoynm

Faces2Keywords has been tested with:

  • Microsoft Windows XP
  • Google Picasa 3.8
  • Adobe Photoshop Lightroom 3.0
  • Lua 5.1.4

Usage Instructions

From Lightroom, select the menu Library > Plug-in Extras > Import from Folder of Selected Photo, under the item Faces2Keywords.

Faces2Keywords will then:

  1. Create Keywords for each Picasa contact found in the Contacts.xml file (if not already created)
  2. Locate the folder of the selected photo
  3. Read the file .picasa.ini in that folder, with contains Picasa's "faces" metadata
  4. Tag the photos in that folder with Keywords for each contact, as found by Picasa's "face detection" feature
